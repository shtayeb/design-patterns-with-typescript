## Local Setup

#

Install TypeScript globally

```shell
npm i typescript -g
```

Compile and watch the typescript files

```shell
tsc -w
```

To run the JS files you need to have Node-js installed

```shell
cd build
```

```shell
node 2_observer_pattern.js
```
